# Aqua Riser -VRジェットパック-

**【Meta Quest 専用】**
「噴き上がる水圧で空を舞え！」
浮遊感と物理演算にこだわったMeta Quest向けVRジェットパック・アクションゲームです。

 **[Webサイト（オンラインランキング・ダウンロードはこちら）](https://yuga-soy.github.io/AquaRiser-VR/)**

##  ゲーム概要
海面スレスレを滑空し、リングを潜り抜けてスコアを競うタイムアタックVRゲームです。
流体抵抗や指向性地面効果などの物理シミュレーションを取り入れ、独特の浮遊感と操作感を実現しています。

###  主な特徴
* **リアルな物理挙動**: 水面スレスレを飛ぶことで強力な浮力を得る「地面効果」を再現。
* **オンラインマルチプレイ**: Photon (PUN2) を使用した、1vs1の白熱したリアルタイムレース！
* **グローバルランキング**: Firebaseを連携し、世界中のプレイヤーとスコアやオンライン人数をリアルタイムに同期。

##  操作方法 (Meta Quest コントローラー)
* **[L/R] Index Trigger (人差し指)** : 左右のジェット噴射
* 腕の向きと傾きで、推力の方向を直感的にコントロールします。

## 動作環境・インストール方法
本作は **Meta Quest (2 / Pro / 3)** のスタンドアローン動作専用アプリ（`.apk`）です。

1. 本リポジトリの [Releases](../../releases) ページ、または[公式Webサイト](https://yuga-soy.github.io/AquaRiser-VR/)から最新の `AquaRiser_v1.0.apk` をダウンロードしてください。
2. [SideQuest](https://sidequestvr.com/) や Meta Quest Developer Hub 等を使用して、Meta Quest端末へAPKファイルをインストール（サイドロード）してください。
3. Quest内の「提供元不明のアプリ」からゲームを起動できます。

## 🛠️ 開発環境 / 使用技術
